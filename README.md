# Product Page Control

Projeto simples com os arquivos:

- `product-page-control.html` — página do produto
- `style-control.css` — estilos

Como usar:

1. Abra `product-page-control.html` no navegador.
2. Edite os arquivos conforme necessário.

Como subir para o GitHub (exemplo):

```bash
cd "c:\\Users\\Aluno\\Downloads\\product-page-control"
git init
git add .
git commit -m "Initial commit"
# no GitHub: criar um repo vazio e copiar a URL
git remote add origin https://github.com/SEU_USUARIO/NOME_REPO.git
git branch -M main
git push -u origin main
```

Se precisar, posso fazer o push por você se fornecer a URL do repositório remoto (ou autorizar via token).
